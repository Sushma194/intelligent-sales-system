📈 Intelligent Sales Forecasting & Inventory Optimization System
🚀 Project Overview

This project is an AI-powered Business Intelligence Dashboard built using Streamlit, Python, and Machine Learning.
It helps in analyzing sales data, forecasting future sales, optimizing inventory, and generating business reports.

🎯 Features
📂 Data Module
CSV File Upload
Live Dataset Preview
Automatic Data Storage in Session
🧹 Data Preprocessing
Missing Value Handling
Duplicate Removal
Data Cleaning Flow Chart
Clean vs Raw Data Comparison
📊 EDA Analysis
Sales Trend Analysis
Revenue Analysis
Pie Charts & Bar Charts
Scatter Plot Analysis
🤖 Model Training
Linear Regression Model
Actual vs Predicted Graph
Model Accuracy (R² Score)
Error Analysis (MAE)
📈 Sales Forecasting
Future Sales Prediction
Forecast Line Graph
Forecast Table
Historical vs Future Comparison
📦 Inventory Optimization
Reorder Point Calculation
Safety Stock Analysis
Inventory Charts (Bar + Pie)
📄 Reports Module
Business Report Generation
KPI Summary Cards
Dataset Preview
Graphical Report View
Download CSV Report
Flow Chart of System Pipeline

▶️ How to Run Project
1. Clone Repository
git clone https://github.com/Sushma194/intelligent-sales-system.git
2. Install Dependencies
pip install -r requirements.txt
3. Run Application
streamlit run app.py
